# Full Stack AI Career Coach with Next JS, Neon DB, Tailwind, Prisma, Inngest, Shadcn UI

### SahyogAI

A full-stack AI-powered career guidance platform that helps users explore career paths, get personalized recommendations, and improve decision-making using intelligent insights.

Features:
 Authentication with Clerk
 AI-powered career recommendations using Gemini API
 Personalized onboarding and career insights
 Modern UI built with Tailwind CSS and shadcn/ui
 Background job handling with Inngest
 Scalable database integration with Neon (PostgreSQL)
 ORM management using Prisma
```

Teck Stack:
Frontend: Next.js, Tailwind CSS, shadcn/ui
Backend: Next.js API routes
Database: Neon (PostgreSQL)
ORM: Prisma
Authentication: Clerk
AI Integration: Gemini API
Background Jobs: Inngest
```
